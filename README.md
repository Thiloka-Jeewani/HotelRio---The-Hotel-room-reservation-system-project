## Overview

LuxStay is a Java-based Tourism Management Web Application developed using JSP, Servlets, Maven, HTML, CSS, and JavaScript. The system provides user authentication, profile management, and password recovery functionalities while applying Object-Oriented Programming (OOP) concepts and CRUD operations.This is a group project and i here developed user management part.

## Features

* User Registration
* User Login and Authentication
* User Profile Management
* Forgot Password with OTP Verification
* Session Management
* Role-Based Access Control
* File-Based Data Storage
* Responsive User Interface

## Technologies Used

* Java
* JSP (Java Server Pages)
* Servlets
* Maven
* HTML5
* CSS3
* JavaScript
* Apache Tomcat

## Installation

### Prerequisites

* Java JDK 17 or later
* Apache Maven
* Apache Tomcat 10

### Steps

1. Clone the repository.
2. Open the project in IntelliJ IDEA or VS Code.
3. Configure Apache Tomcat 10.
4. Build the project:

```bash
mvn clean package
```

5. Deploy the generated WAR file to Tomcat.
6. Access the application through:

```text
http://localhost:8080/HotelRio
```
